# C-Clamp Design and Analysis using ABAQUS

## Objective
To design and analyze a C-Clamp using ABAQUS and evaluate its structural performance under applied clamping loads using Finite Element Analysis (FEA).

## Software Used
- ABAQUS

## Project Overview
- Created a 3D model of the C-Clamp.
- Assigned appropriate material properties.
- Applied clamping load and boundary conditions.
- Generated the finite element mesh.
- Performed static stress analysis.
- Evaluated stress distribution and deformation.

## Results
- Successfully analyzed the C-Clamp under loading.
- Identified regions of maximum stress concentration.
- Verified the structural strength and deformation behavior.

## Skills Demonstrated
- Finite Element Analysis (FEA)
- Stress Analysis
- ABAQUS
- Mechanical Component Design
- Structural Simulation.
